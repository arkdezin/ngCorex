---

## Supported Versions

ngCorex is currently in **early development**.

Security fixes will be applied to:

- The latest published version

Older versions may not receive patches.

---

## Disclosure Process

Once a vulnerability is reported:

1. We will acknowledge receipt
2. Investigate the issue
3. Prepare a fix if necessary
4. Publish an update and disclosure if appropriate

Thank you for helping keep ngCorex secure.
